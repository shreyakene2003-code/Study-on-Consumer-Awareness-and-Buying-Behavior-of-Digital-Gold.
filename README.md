# Study-on-Consumer-Awareness-and-Buying-Behavior-of-Digital-Gold.
Analyzed consumer awareness and buying behavior of digital gold using data collected across demographics. Performed data cleaning and analysis in Excel using Pivot Tables and formulas, and built an interactive dashboard in Power BI to visualize key insights such as awareness levels, platform usage, and investment patterns. 
 Project Overview

This project analyzes consumer awareness and buying behavior towards digital gold as an investment option. With the rise of digital platforms like Google Pay, Paytm, and PhonePe, digital gold is becoming increasingly popular among investors.

The project uses Excel for data cleaning and analysis, and Power BI for building an interactive dashboard to generate business insights.

 Objectives
To measure consumer awareness of digital gold
To analyze buying behavior and investment patterns
To identify factors influencing purchase decisions
To study platform preferences and trust levels
 Dataset
Simulated dataset with 300+ records
Includes demographics, awareness, investment behavior, and user preferences
Contains missing values and inconsistencies for real-world analysis practice
Tools & Technologies
Microsoft Excel (Data Cleaning, Pivot Tables, Charts)
Power BI (Dashboard & Data Visualization)
 Key Analysis Performed
Data cleaning and preprocessing
Pivot Table analysis (awareness, income vs investment, platform usage)
Trend analysis and comparison
KPI calculations (Total Users, Awareness %, Avg Investment)
Dashboard Features
KPI Cards (Total Users, Awareness %, Average Investment)
Bar Chart (Platform Usage)
Pie Chart (Preferred Investment Type)
Line Chart (Purchase Trends)
Slicers (Age, Income, Platform)
 Key Insights
Younger users show higher awareness of digital gold
Digital platforms are the preferred medium for investment
Higher income groups tend to invest more
Trust level strongly impacts purchase decisions
 Conclusion

The project highlights growing interest in digital gold and shows how awareness, trust, and convenience influence user behavior. These insights can help businesses improve marketing strategies and increase adoption.
